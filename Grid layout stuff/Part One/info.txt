 A grid container is an HTML element that serves as the parent or container 
 for defining a grid layout using CSS Grid. 
 It establishes a grid formatting context and contains grid items. 
 
 Grid items are located inside the container and make up what's inside the container. 
 They can be changed to create different grid layouts.


 Grid lines are horizontal and vertical lines that divide the grid into rows and collums
 they are used as reference points for positioning the grid items and defining the layout
 grid lines are numbered starting from 1

 Grid cells are the spaces between two adjacent horizontal and vertical grid lines
 each cell can contain one or more grid items and teh size of the cells are 
 determined by the grids the colum and row definitions
